Данный сайт нацелен на создание цифровизированной карты стоматологического больного для возможность визуализации записи зубной формулы, автоматической индексной оценки состояния тканей пародонта и твердых тканей зубов. Он был реализован на основе фреймворка Django с использованием библиотек html2canvas и jsPDF, которые позволяют скачать полученный документ в формате pdf. Для построения графиков я воспользовалась библиотекой Chart.js.
